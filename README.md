# KingdomsAttributes

Дополнительный плагин для KingdomsAddon - модификатор размера для игроков снежного королевства.

## Функционал

- Игроки снежного королевства получают изменённый `GENERIC_SCALE` (по умолчанию 0.8)
- Список исключений - игроки, которым не применяется модификатор
- Админы (`kingdoms.admin`) не получают модификатор

## Конфигурация

```yaml
snow-kingdom:
  # Множитель размера (1.0 = 100%, 0.8 = 80%)
  scale: 0.8
  
  # Игроки-исключения (ники нечувствительны к регистру)
  excluded-players:
    - ExamplePlayer1
    - ExamplePlayer2
```

## Требования

- Folia 1.21.4+ или Paper 1.21+
- Java 21+
- KingdomsAddon

## Установка

1. Скачайте JAR файл
2. Поместите в папку `plugins/`
3. Убедитесь, что KingdomsAddon установлен
4. Перезапустите сервер
5. Настройте `config.yml`

## Сборка

```bash
mvn clean package
```

## Folia-совместимость

Плагин использует `Entity#getScheduler()` для безопасной работы с атрибутами в многопоточной среде Folia.
